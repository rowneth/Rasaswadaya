# Rasasvadya.lk

The ultimate Sri Lankan entertainment hub. Discover university events, cultural shows, musical concerts, and more.

## Features

- **Event Discovery**: Find events happening in and around Sri Lankan universities.
- **User Accounts**: Create a profile to personalize your experience.
- **Artist Profiles**: Artists can create specialized profiles to showcase their work (Music, Dance, etc.).

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `src/app/page.tsx`: Landing page.
- `src/app/login/page.tsx`: Login page.
- `src/app/signup/page.tsx`: Signup page with User/Artist toggle.
- `src/app/globals.css`: Global styles and theme configuration.

## Technologies

- Next.js
- TypeScript
- Tailwind CSS
- Lucide React (Icons)
